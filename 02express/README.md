# 📦 Product Management System

Sebuah aplikasi web responsif untuk manajemen inventaris dan produk, dibangun menggunakan Node.js, Express, dan EJS Templating Engine. Proyek ini menerapkan pendekatan modular pada sisi *frontend* (DRY - *Don't Repeat Yourself*) untuk memastikan kode yang bersih dan mudah dikelola.

## ✨ Fitur Utama

* **Manajemen Produk (CRUD):** Antarmuka intuitif untuk mengelola data produk meliputi Nama, Harga, Kategori, dan Foto.
* **Sistem Templating Modular:** Menggunakan EJS Partials (`_head`, `_foot`, `_nav`, `_message`) untuk efisiensi penulisan struktur HTML.
* **Notifikasi Real-time (Flash Messages):** Dilengkapi dengan alert interaktif untuk memberikan *feedback* kepada pengguna saat aksi (seperti tambah/hapus data) berhasil dilakukan.
* **Paginasi Data:** Navigasi tabel yang terstruktur untuk menangani volume data produk yang besar.
* **UI/UX Responsif:** Tampilan antarmuka yang modern dan optimal di berbagai ukuran perangkat, ditenagai oleh Bootstrap 5.

## 🛠️ Teknologi yang Digunakan

* **Backend:** Node.js, Express.js
* **View Engine:** EJS (Embedded JavaScript)
* **Frontend Library:** Bootstrap v5.3.2, FontAwesome v6.4.2
* **Fitur Tambahan:** Express Session & Flash (untuk notifikasi)
